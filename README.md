# Selenium UI Tests - Coffee Cart App

This project contains automated UI tests for the [CoffeeCart](https://coffee-cart.app/) web application, created using:

- **Java 21**
- **Selenium WebDriver 4.34.0**
- **JUnit Jupiter 6.0.0-M2**
- **WebDriverManager 6.2.0**

It's designed as a playground for learning and practicing UI automation testing in Java.
